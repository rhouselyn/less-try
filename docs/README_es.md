## ¿Qué es 呱邻国?

呱邻国 es una plataforma inmersiva de aprendizaje de idiomas extranjeros impulsada por IA. Proporcionas cualquier texto y la IA genera automáticamente un vocabulario, traducción por oraciones y múltiples ejercicios, junto con lectura en voz alta, convirtiendo cada texto en tu material de estudio personalizado.

**Cualquier idioma → cualquier idioma, tú decides el material.**

**Solo necesitas una API Key, todo impulsado por la capacidad pura de los LLM.**

## Lo que Duolingo no puede hacer, 呱邻国 sí lo hace

| Las limitaciones de Duolingo | La solución de 呱邻国 |
|-------------|-------------|
| **No hay lista de vocabulario, no hay forma de repasar** | Genera automáticamente un vocabulario completo, con índice alfabético, búsqueda y detalles palabra por palabra; consulta en cualquier momento la definición, fonética, variaciones morfológicas y oraciones de ejemplo de cada palabra |
| **Quieres buscar otras palabras mientras haces ejercicios** | Abre la tabla de vocabulario en cualquier momento durante el aprendizaje, consulta la definición y detalles de cualquier palabra sin interrumpir el ritmo de estudio |
| **Aprendes pero difícilmente puedes aplicarlo** | Aprendes del material que proporcionas: letras de canciones, noticias, diálogos, artículos académicos; lo que aprendes es lo que realmente encontrarás |
| **No soporta idiomas minoritarios** | Soporta aprendizaje mutuo entre cualquier idioma, la IA detecta automáticamente el idioma, lectura TTS en más de 120 idiomas, sin limitaciones por los recursos de la plataforma |
| **No puedes profundizar en la comprensión de un artículo** | Ingresa el artículo y la IA traduce por oraciones, extrae todo el vocabulario y genera ejercicios; desde palabras hasta oraciones hasta el artículo completo, domina a fondo cada texto |

## ¿Cómo satisface tus necesidades?

### 🎯 "Tengo material, quiero aprender directamente" → Modo de entrada directa

Pega un artículo, una letra de canción, una noticia, un diálogo de serie — cualquier texto en idioma extranjero que ingreses, la IA detecta automáticamente el idioma, traduce por oraciones, extrae vocabulario y personaliza el contenido de aprendizaje para ti.

### 🌐 "Quiero usar material en mi idioma materno para aprender un idioma extranjero" → Modo de traducción y aprendizaje

Ingresa un texto en tu idioma materno, la IA lo traduce al idioma que deseas aprender y luego genera vocabulario y ejercicios basados en el texto traducido. El material que ya conoces se convierte en tu punto de partida para aprender un idioma extranjero.

### ✨ "No tengo material, genéralo para mí" → Modo de generación por IA

Dile a la IA qué tema quieres aprender (por ejemplo, "diálogos cotidianos para pedir café", "escenarios de pedir direcciones en viaje") y la IA generará automáticamente un texto en el idioma objetivo, para luego comenzar a aprender. Puedes aprender incluso sin material previo.

### 🗣️ "Quiero practicar la comprensión auditiva" → Lectura en voz alta

Basado en TTS nativo del navegador, soporta más de 120 idiomas. Tanto palabras como oraciones pueden ser leídas en voz alta, con cambio libre entre velocidad normal y lenta. La fase dos también incluye ejercicios especiales de comprensión auditiva.

### 🌍 "Mi idioma materno no es inglés ni chino" → Interfaz en cualquier idioma

Selecciona tu idioma materno y toda la interfaz se traduce automáticamente a ese idioma. Sin importar cuál sea tu idioma materno, podrás usar la aplicación sin barreras.

### 📖 "Comprensión lectora de secundaria, quiero dominar completamente un artículo" → Modo de lectura intensiva

Ingresa el artículo de comprensión lectora y la IA traduce automáticamente por oraciones, extrae todas las palabras y frases nuevas, genera tarjetas de vocabulario y múltiples ejercicios. Desde la comprensión palabra por palabra hasta el dominio del artículo completo, te ayuda a entender a fondo cada detalle del texto.

## Sistema de aprendizaje

Dos fases de aprendizaje + repaso de errores, dominando cada punto de conocimiento de manera progresiva:

| Fase | Contenido | Tipos de ejercicios |
|------|------|------|
| **Fase 1 · Reconocimiento de vocabulario** | Tarjetas de palabras + Traducción de oraciones | Selección de palabras, traducción de oraciones |
| **Fase 2 · Entrenamiento integral** | Comprensión auditiva + Rellenar espacios + Reorganización | Comprensión auditiva, rellenar espacios con palabras ocultas, reorganización de traducción |

Cada unidad tiene 10 preguntas; al completarla obtienes una evaluación con ⭐ estrellas. Las preguntas incorrectas entran automáticamente al repaso de errores hasta que las domines.

## 🚀 Inicio rápido

### Opción 1: Descargar la aplicación de escritorio (Recomendado)

Ve a [GitHub Releases](https://github.com/rhouselyn/Gualingo/releases) para descargar el instalador de tu plataforma:

| Plataforma | Archivo |
|------|------|
| Windows | `Gualingo-Windows.zip` |
| macOS | `Gualingo-macOS.tar.gz` |
| Linux | `Gualingo-Linux.tar.gz` |

Extrae y ejecuta `Gualingo` — no requiere instalación de Python o Node.js.

### Opción 2: Docker

```bash
docker run -d \
  -p 8000:8000 \
  -v gualingo-data:/root/.local/share/Gualingo \
  ghcr.io/rhouselyn/gualingo:latest
```

Abre http://localhost:8000 para comenzar a usar.

### Opción 3: Ejecutar desde el código fuente

#### Requisitos del entorno

- Python 3.10+
- Node.js 18+
- Una LLM API Key (soporta interfaces compatibles con OpenAI, como SiliconFlow, DeepSeek, etc.)

#### Instalación e inicio

```bash
# 1. Instalar dependencias del backend
cd backend
pip install -r requirements.txt

# 2. Iniciar el backend
uvicorn main:app --reload --host 0.0.0.0 --port 8000

# 3. Instalar dependencias del frontend (elige un estilo de frontend)
cd ../frontend-soft-ui
npm install

# 4. Iniciar el frontend
npm run dev
```

Abre http://localhost:5174, haz clic en ⚙️ Configuración en la esquina superior derecha e ingresa tu API Key para comenzar a aprender.

#### Modo de aplicación de escritorio

```bash
# Instalar dependencias de la aplicación de escritorio
pip install pywebview

# Construir frontend-soft-ui
cd frontend-soft-ui
npm install
npm run build
cd ..

# Iniciar la aplicación de escritorio
python app.py
```

## 📖 Flujo de uso

```
Ingresar texto → Traducción por oraciones por IA → Generar vocabulario → Fase 1: Aprender palabras → Fase 2: Practicar oraciones → Repaso de errores
```

1. **Ingresar texto**: Pega directamente, traduce al idioma objetivo, o deja que la IA lo genere
2. **Explorar el diccionario**: Consulta la traducción por oraciones y las definiciones del vocabulario, revisa cualquier palabra en cualquier momento
3. **Fase 1**: Selección de palabras, traducción de oraciones, comprensión auditiva
4. **Fase 2**: Rellenar espacios con palabras ocultas, reorganización de traducción
5. **Repaso de errores**: Las preguntas incorrectas se recopilan automáticamente, práctica de refuerzo hasta dominarlas

## ⚙️ Configuración

Toda la configuración se realiza desde la interfaz, sin necesidad de editar archivos de configuración:

- **API Key**: Soporta rotación de múltiples configuraciones, cambio automático ante limitación de velocidad
- **Idioma materno**: Selecciona el idioma de visualización de la interfaz
- **Cantidad por página**: Controla la cantidad de palabras mostradas por página en el vocabulario
- **Intervalo de reintento**: Tiempo de espera después de la limitación de velocidad de la API
