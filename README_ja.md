## 呱邻国とは？

呱邻国は、AI 駆動の没入型外国語学習プラットフォームです。あなたが任意のテキストを提供すると、AI が自動的に語彙表、文ごとの翻訳、多様な練習問題を生成し、音声読み上げと組み合わせて、あらゆる文章をあなた専用の学習教材に変えます。

**どんな言語 → どんな言語へ、教材はあなた次第。**

**API キー一つだけで、データベース不要、純粋な LLM の力ですべてを駆動。**

## デュオリンゴにできないことを、呱邻国は実現

| デュオリンゴの課題 | 呱邻国の解決策 |
|-------------|-------------|
| **単語リストがなく、復習の手段がない** | 完全な語彙表を自動生成、アルファベット索引、検索、単語ごとの詳細に対応。いつでも各単語の意味、発音記号、語形変化、例文を参照可能 |
| **問題を解きながら他の単語を調べたい** | 学習中にいつでも単語サブテーブルを開き、任意の単語の意味と詳細を確認。学習の流れを止めない |
| **学んでもなかなか実践で使えない** | あなたが提供した素材をそのまま学習——歌詞、ニュース、セリフ、論文、学んだことが実際に出会うものそのもの |
| **マイナー言語に対応していない** | 任意の言語間の学習に対応、AI が言語を自動検出、120+ 言語の TTS 読み上げ、プラットフォームのリソース制限を受けない |
| **一つの記事を深く理解できない** | 記事を投げ込めば、AI が文ごとに翻訳、全語彙を抽出、練習問題を生成。単語から文、記事全体まで、毎篇文章を完全に理解 |

## あなたのニーズにどう応えるか？

### 🎯 「素材があるから、そのまま学びたい」→ 直接入力モード

記事、歌詞、ニュース、セリフ——任意の外国語テキストを貼り付ければ、AI が言語を自動検出、文ごとに翻訳、語彙を抽出し、あなたに合わせた学習コンテンツを作成。

### 🌐 「母語の素材を使って外国語を学びたい」→ 翻訳学習モード

母語のテキストを入力すると、AI が学びたい言語に翻訳し、翻訳後のテキストに基づいて語彙と練習を生成。おなじみの素材が、外国語学習の出発点に。

### ✨ 「素材がない、生成してほしい」→ AI 生成モード

学びたいテーマを AI に伝える（例：「日常のカフェ注文の会話」「旅行での道尋ねシーン」）と、AI がターゲット言語のテキストを自動生成し、そのまま学習を開始。素材がなくても学べる。

### 🗣️ 「リスニングを練習したい」→ 音声読み上げ

ブラウザネイティブの TTS に基づき、120+ 言語に対応。単語も文も読み上げ可能、通常速度/スロー速度を自由に切り替え。フェーズ2には専用のリスニング理解練習もある。

### 🌍 「母語が英語/中国語ではない」→ 任意言語インターフェース

あなたの母語を選ぶと、インターフェース全体がその言語に自動翻訳。母語が何であっても、障壁なく使える。

### 📖 「中学校の読解問題、記事を完全に理解したい」→ 文章精読モード

読解問題の記事を入力すると、AI が文ごとに翻訳、すべての未知の単語やフレーズを抽出、語彙カードと多様な練習問題を生成。単語ごとの理解から記事全体の把握まで、記事のあらゆる詳細を完全に理解できる。

## 学習体系

2フェーズ学習 + 間違いレビューで、段階的に各知識ポイントを習得：

| フェーズ | 内容 | 問題タイプ |
|------|------|------|
| **フェーズ1・語彙認知** | 単語カード + 文翻訳 | 単語選択、文翻訳 |
| **フェーズ2・総合訓練** | リスニング + 穴埋め + 並べ替え | リスニング理解、穴埋め、翻訳並べ替え |

各ユニット10問、完了後に ⭐ スター評価を獲得。間違えた問題は自動的に間違いレビューに追加され、習得するまで繰り返し練習。

### フェーズ1・語彙認知

| 問題タイプ | スクリーンショット | 説明 |
|------|------|------|
| 単語選択 | ![単語選択](docs/screenshots/vocab.png) | 四択問題、単語を見て意味を選択、音声読み上げと発音記号付き |
| 文翻訳 | ![文翻訳](docs/screenshots/translate%20t2s.png) | ソース言語の文を見て、母語の単語から翻訳を組み立てる |
| リスニング理解 | ![リスニング問題](docs/screenshots/listening.png) | 文を聞いて、単語から聞こえた内容を組み立てる。通常速度/スロー速度切り替え対応 |

### フェーズ2・総合訓練

| 問題タイプ | スクリーンショット | 説明 |
|------|------|------|
| 穴埋め | ![穴埋め問題](docs/screenshots/mask.png) | 文のキーワードが空白になり、選択肢から正解を選ぶ |
| 翻訳並べ替え | ![翻訳並べ替え](docs/screenshots/translate%20s2t.png) | 母語の翻訳を見て、ターゲット言語の単語から元の文を復元する |

## 主なページ

### ホーム & 入力

![ホーム](docs/screenshots/main.png)

3つの入力モードで、すべての学習シーンをカバー：
- **直接入力** — 外国語テキストを貼り付け、AI が言語を自動検出、文ごとに翻訳、語彙を抽出
- **自動翻訳** — 母語テキストを入力、AI がターゲット言語に翻訳してから学習
- **自由生成** — 学びたいテーマを AI に伝えると、学習コンテンツを自動生成

### 辞書ページ

![辞書ページ](docs/screenshots/contrast.png)

左側に文翻訳、右側に語彙表、下線付きの単語をクリックすると詳細を表示。アルファベット索引で素早く検索、各単語に発音記号、品詞、意味、語形変化、例文付き。

![単語詳細](docs/screenshots/contrast_detail.png)

単語詳細は再生成に対応、記憶補助で連想記憶をサポート。

### 学習ユニット

![学習ユニット](docs/screenshots/units.png)

2フェーズ学習体系、ユニットが段階的にアンロック、完了後にスター評価を獲得。「新規単語のみ学習」と「リスニングスキップ」スイッチに対応、学習ペースを自由にカスタマイズ。

## 🚀 クイックスタート

### 環境要件

- Python 3.10+
- Node.js 18+
- LLM API キー（OpenAI 互換インターフェース対応、SiliconFlow、DeepSeek など）

### インストールと起動

```bash
# 1. バックエンドの依存関係をインストール
cd backend
pip install -r requirements.txt

# 2. バックエンドを起動
uvicorn main:app --reload --host 0.0.0.0 --port 8000

# 3. フロントエンドの依存関係をインストール
cd ../frontend
npm install

# 4. フロントエンドを起動
npm run dev
```

http://localhost:5173 を開き、右上の ⚙️ 設定から API キーを入力すれば、学習を開始できます。

## 📖 利用フロー

```
テキスト入力 → AI 文ごと翻訳 → 語彙表生成 → フェーズ1：単語学習 → フェーズ2：文練習 → 間違いレビュー
```

1. **テキスト入力**：直接貼り付け、ターゲット言語に翻訳、または AI に生成させる
2. **辞書閲覧**：文ごとの翻訳と語彙の意味を確認、いつでも任意の単語を参照
3. **フェーズ1**：単語選択、文翻訳、リスニング理解
4. **フェーズ2**：穴埋め、翻訳並べ替え
5. **間違いレビュー**：間違えた問題を自動収集、習得するまで強化練習

---

## ⚙️ 設定

すべての設定はインターフェースから完了、設定ファイルの編集不要：

- **API キー**：複数設定のローテーションに対応、レート制限時に自動切り替え
- **母語**：インターフェースの表示言語を選択
- **ページあたりの数量**：語彙表の1ページあたりの単語数を制御
- **リトライ間隔**：API レート制限後の待機時間
