# 少邻国 (Lesslingo) - 核心功能重构 - 验证检查清单

- [ ] 自动分句功能：输入包含多个句子的文本，系统能正确切分为多个句子
- [ ] 词汇提取和去重：输入包含重复单词的文本，提取的单词列表中每个单词只出现一次
- [ ] Tool Schema 实现：LLM 返回的数据符合 Tool Schema 定义
- [ ] 结构化输出：输出包含所有必要字段（翻译、分词、音标、释义、变体、例句、选项、语法讲解）
- [ ] 双栏字典 UI：界面布局清晰，双栏结构合理，左侧单词列表，右侧详细信息
- [ ] 视觉设计：符合 Anthropic 风格，简约现代
- [ ] 自动跳转功能：点击生成按钮后，处理完成自动跳转到单词表页面
- [ ] 加载状态：跳转过程中显示适当的加载状态
- [ ] API 响应：所有 API 端点正常响应，响应时间合理
- [ ] 品牌一致性：界面设计符合 Anthropic 品牌指南
- [ ] 视觉设计优化：配色方案和谐，排版清晰，间距合理
- [ ] 动画效果：流畅自然的动画和过渡效果
- [ ] 交互反馈：及时明确的交互反馈
- [ ] 整体用户体验：流畅，无明显卡顿，能处理各种输入场景
