# 少邻国 (Lesslingo) 项目实施计划

## 1. 项目概述

**少邻国** 是一款纯本地运行的网页版外语学习系统，以 Agent 为核心，主打"所有生成与学习内容均由 AI 现场实时生成"，无需预先收集任何语言数据库。

### 核心特性
- 完全本地化：无注册、无云端、无账号，进度、prompt、上传文件均以本地 JSON + 文件形式存储
- 支持多模态输入：文本、音频（转文本）、图片（OCR 转文本）
- 学习语言与母语自由选择，AI 自动翻译、对齐、拆解
- 风格：类似 Anthropic 官网的简约动效风
- AI 渠道：NVIDIA API
  - 文本生成 / Agent：`minimax-m2.7`（支持 Tool Use）
  - 语音识别（ASR）：`whisper-large-v3`（音频转文本 + 纠错）
  - 语音合成（TTS）：`magpie-tts-multilingual`（支持多语言）
  - OCR：`nemotron-ocr-v1`

## 4. 技术实现要点

### 4.1 Tool Call 规范
- **词汇生成 Tool**：每10个词一组，生成符合原文的释义、音标、变体、两个意思相同的例句（temp=0）
- **题目生成 Tool**：生成4选项选择题，包含原文干扰词
- **干扰词生成 Tool**：专门生成读音相近的干扰词（听力题）
- **句子处理 Tool**：处理句子时生成4个合理的冗余tokens（写题型）

### 4.2 性能优化
- TTS 生成间隔：1-2秒随机时长
- 词汇生成：chunk=10并带上下文
- 题目快照：防止题目漂移
- 断点续传：支持TTS和其他长时间任务

### 4.3 防坑指南
- **题目漂移**：所有LLM生成内容存入snapshot.json
- **API限速**：严格控制调用频率
- **认字选项与详情生成分离**：选项生成可适度随机，详情页temp=0
- **第二阶段动态循环**：实时判断可组成的新句子
- **UI防作弊**：下一题按钮仅在回顾旧模块时出现
