
# Lesslingo 功能改进计划

## 1. 问题分析

### 现有问题
1. **单词表显示不完整**：只显示4个单词
2. **练习生成问题**：
   - 选项包含句子，不是纯单词
   - 错误选项可能包含单词本身的意思
   - 掩码token基于自动分词，而非翻译token
   - 短句子无法生成练习
3. **第二阶段练习**：
   - 仅支持8+单词的句子
   - 随机化逻辑需要改进

## 2. 改进计划

### 2.1 后端改进

#### 2.1.1 存储层 (storage.py)
- 无修改需求

#### 2.1.2 文本处理 (text_processor.py)
- 改进 `generate_masked_sentence`：
  - 使用翻译token而非自动分词
  - 支持任意长度句子（<8个token时掩码1个）
  - 集成备选词库
- 添加 `get_fallback_distractors`：提供备选干扰词

#### 2.1.3 主后端 (main.py)
- 改进 `generate_multiple_choice` 调用：
  - 修改prompt，确保选项为纯单词
  - 确保错误选项不是单词本身的意思
- 改进 `get_phase_unit_exercise`：
  - 支持任意长度句子
  - 实现两阶段随机化逻辑

### 2.2 前端改进
- 无重大修改需求，确保与后端新API兼容

## 3. 实施步骤

### 步骤1：改进文本处理模块
- 修改 `generate_masked_sentence` 函数
- 添加备选词库和干扰词生成函数

### 步骤2：更新后端API
- 修改单词选项生成逻辑
- 更新练习生成逻辑

### 步骤3：测试与验证
- 测试长句子处理
- 测试单词表完整性
- 测试练习生成质量
- 测试随机化效果

## 4. 预期结果
- 单词表显示完整
- 练习选项为纯单词，错误选项合理
- 支持任意长度句子的练习
- 随机化效果良好
- 无超时或错误
