## 呱邻국은 무엇인가요?

呱邻국은 AI 기반의 몰입형 외국어 학습 플랫폼입니다. 어떤 텍스트든 제공하면, AI가 자동으로 어휘표, 문장별 번역, 다양한 연습 문제를 생성하고 음성 읽기까지 지원하여, 모든 글을 나만의 학습 자료로 만들어줍니다.

**어떤 언어 → 어떤 언어, 당신의 소재는 당신이 결정합니다.**

**API Key 하나만 있으면 됩니다. 데이터베이스 불필요, 순수 LLM 능력으로 모든 것을 구동합니다.**

## 듀오링고가 하지 못하는 것, 呱邻국이 해냈습니다

| 듀오링고의 한계 | 呱邻국의 해법 |
|-------------|-------------|
| **단어장이 없어 복습 불가** | 완전한 어휘표를 자동 생성, 알파벳 인덱스·검색·단어별 상세 정보를 지원하여 언제든 각 단어의 뜻, 발음 기호, 활용 형태, 예문을查阅할 수 있습니다 |
| **문제 풀면서 다른 단어를 찾고 싶을 때** | 학습 중 언제나 단어 분할표를 열어 어떤 단어든 뜻과 상세 정보를 확인할 수 있어, 학습 흐름이 끊기지 않습니다 |
| **배워도 실제로 사용하기 어려움** | 어떤 소재를 제공하면 그것을 배웁니다—가사, 뉴스, 대사, 논문, 배운 것이 곧 실제로 마주칠 내용입니다 |
| **소수 언어 미지원** | 어떤 언어 간 학습도 지원, AI가 자동으로 언어를 감지하고 120+개 언어 TTS 읽기를 제공하여 플랫폼 자원의 제약을 받지 않습니다 |
| **한 편의 글을 깊이 이해하기 어려움** | 글을 넣으면 AI가 문장별 번역, 전체 어휘 추출, 연습 문제 생성까지—단어에서 문장, 전체 글까지 한 편의 글을 완벽하게 소화합니다 |

## 당신의 필요를 어떻게 충족하나요?

### 🎯 "소재가 있는데 바로 학습하고 싶어요" → 직접 입력 모드

기사, 가사, 뉴스, 대사—어떤 외국어 텍스트든 붙여넣으면 AI가 자동으로 언어를 감지하고, 문장별 번역, 어휘 추출까지 당신만을 위한 학습 콘텐츠를 맞춤 생성합니다.

### 🌐 "모국어 소재로 외국어를 배우고 싶어요" → 번역 학습 모드

모국어 텍스트를 입력하면 AI가 배우고 싶은 언어로 번역한 뒤, 번역된 텍스트를 기반으로 어휘와 연습을 생성합니다. 익숙한 소재가 외국어 학습의 출발점이 됩니다.

### ✨ "소재가 없는데 생성해 주세요" → AI 생성 모드

AI에게 학습하고 싶은 주제를 알려주세요(예: "일상 카페 주문 대화", "여행 길 찾기 상황"), AI가 자동으로 목표 언어 텍스트를 생성한 뒤 학습을 시작합니다. 소재가 없어도 학습할 수 있습니다.

### 🗣️ "듣기 연습을 하고 싶어요" → 음성 읽기

브라우저 네이티브 TTS 기반, 120+개 언어를 지원합니다. 단어와 문장 모두 읽기 가능, 일반 속도/느린 속도 자유 전환. 2단계에서는 전용 듣기 이해 연습도 제공합니다.

### 🌍 "모국어가 영어/중국어가 아닌데요" → 어떤 언어든 인터페이스 지원

모국어를 선택하면 전체 인터페이스가 해당 언어로 자동 번역됩니다. 모국어가 무엇이든 장애 없이 사용할 수 있습니다.

### 📖 "중학생 독해력, 한 편의 글을 완벽하게 이해하고 싶어요" → 정독 모드

독해 지문을 넣으면 AI가 자동으로 문장별 번역, 모든 생단어와 구문을 추출하여 어휘 카드와 다양한 연습 문제를 생성합니다. 단어별 이해에서 전체 파악까지, 한 편의 글의 모든 디테일을 완전히 이해하도록 도와줍니다.

## 학습 체계

2단계 학습 + 오답 복습, 점진적으로 각 지식 포인트를 마스터하세요:

| 단계 | 내용 | 문제 유형 |
|------|------|------|
| **1단계 · 어휘 인지** | 단어 카드 + 문장 번역 | 단어 선택, 문장 번역 |
| **2단계 · 종합 훈련** | 듣기 + 빈칸 채우기 + 재구성 | 듣기 이해, 빈칸 채우기, 번역 재구성 |

각 유닛당 10문항, 완료 후 ⭐ 별점 평가를 받으며, 틀린 문제는 자동으로 오답 복습에 들어가 마스터할 때까지 반복합니다.

## 주요 페이지

### 메인 & 입력

![메인](docs/screenshots/main.png)

세 가지 입력 모드로 모든 학습 상황을 커버:
- **직접 입력** — 외국어 텍스트를 붙여넣으면 AI가 자동으로 언어 감지, 문장별 번역, 어휘 추출
- **자동 번역** — 모국어 텍스트를 입력하면 AI가 목표 언어로 번역 후 학습
- **자유 생성** — AI에게 학습하고 싶은 주제를 알려주면 자동으로 학습 콘텐츠 생성

### 사전 페이지

![사전 페이지](docs/screenshots/contrast.png)

왼쪽 문장 번역, 오른쪽 어휘표, 밑줄 친 단어를 클릭하면 상세 정보를 확인할 수 있습니다. 알파벳 인덱스로 빠르게 탐색, 각 단어마다 발음 기호, 품사, 뜻, 활용 형태, 예문 제공.

![단어 상세](docs/screenshots/contrast_detail.png)

단어 상세 정보는 새로고침으로 재생성 가능, 기억 보조로 연상 기억을 돕습니다.

![학습 유닛](docs/screenshots/units.png)

2단계 학습 체계, 유닛이 점진적으로 잠금 해제되며 완료 후 별점 평가를 받습니다. "새 단어만 학습" 및 "듣기 건너뛰기" 스위치를 지원하여 필요에 따라 학습 속도를 맞춤 설정.

## 🚀 빠른 시작

### 환경 요구사항

- Python 3.10+
- Node.js 18+
- LLM API Key 하나 (OpenAI 호환 인터페이스 지원, 예: SiliconFlow, DeepSeek 등)

### 설치 및 실행

```bash
# 1. 백엔드 의존성 설치
cd backend
pip install -r requirements.txt

# 2. 백엔드 실행
uvicorn main:app --reload --host 0.0.0.0 --port 8000

# 3. 프론트엔드 의존성 설치
cd ../frontend
npm install

# 4. 프론트엔드 실행
npm run dev
```

http://localhost:5173 을 열고, 우측 상단 ⚙️ 설정에서 API Key를 입력하면 학습을 시작할 수 있습니다.

## 📖 사용 흐름

```
텍스트 입력 → AI 문장별 번역 → 어휘표 생성 → 1단계: 단어 학습 → 2단계: 문장 연습 → 오답 복습
```

1. **텍스트 입력**: 직접 붙여넣기, 목표 언어로 번역, 또는 AI가 생성
2. **사전 탐색**: 문장별 번역과 어휘 뜻을 확인, 언제든 어떤 단어든查阅
3. **1단계**: 단어 선택, 문장 번역, 듣기 이해
4. **2단계**: 빈칸 채우기, 번역 재구성
5. **오답 복습**: 틀린 문제가 자동 수집되어 마스터할 때까지 강화 연습

## ⚙️ 설정

모든 설정은 인터페이스에서 완료되며, 설정 파일 편집 불필요:

- **API Key**: 여러 그룹 설정 로테이션 지원, 속도 제한 시 자동 전환
- **모국어**: 인터페이스 표시 언어 선택
- **페이지당 수량**: 어휘표 페이지당 표시할 단어 수 제어
- **재시도 간격**: API 속도 제한 후 대기 시간
